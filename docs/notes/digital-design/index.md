# 数字设计与端侧推理

这里收录从个人 Obsidian 学习笔记中整理出的、适合公开阅读的技术文章。内容以日常验证、接口设计和端侧推理实现中的基础问题为主；文章在发布前重新检查了术语、示例和边界条件。

目前包括：

- SystemVerilog Assertion 的时序表达与常用模板；
- 从约束、断言到覆盖率的形式验证工作流；
- CRC、汉明码与游程编码的定位和使用边界；
- 将卷积转换为 GEMM 时的 im2col 数据布局与量化注意事项。
